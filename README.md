

# 2026 复旦大学电子设计竞赛 (A题：精密电流源设计) - 主控固件

## 🏆 项目简介
本项目为 2026 年复旦大学电子设计竞赛 A 题（精密电流源）的数字控制端固件。
系统采用“纯硬件运放闭环线性稳流”作为模拟核心架构，数字控制端创造性地采用了 **STC AI8051U (32位扩展8051)** 搭配 **普中51实验板** 进行魔改化开发。利用 AI8051U 强大的运算能力（42MHz主频）和超大内存（34KB SRAM），完美实现了题设要求的“四位数码管设定”、“电流曲线描绘”以及多点高精度数字校准。

## 🛠 硬件架构与选型
*   **MCU 主控**：STC AI8051U-34K64-PDIP40（**注意：需将普中板原有的 STC89C52RC 拔下，原位替换插入此芯片**）
*   **开发底板**：普中 51-单核-A2/A3/A4 实验板（利用板载 CH340、独立按键、数码管模块）
*   **显示模块**：
    *   板载 4 位动态数码管（显示电流设定值）
    *   外挂 0.96 寸 I2C OLED 屏幕（显示电流实测值，并利用 MCU 大内存实时绘制电流波动曲线）
*   **模拟前段 (外挂模块)**：
    *   **DAC 给定**：DAC8562（16位双路 SPI），负责输出高精度参考电压给运放。
    *   **ADC 采集**：ADS1115（16位四路 I2C），负责采集 0.1Ω 低温漂合金检流电阻两端的电压差。
    *   **核心功率级**：OPA2333精密运放 + IRF540N 功率MOS管（工作于放大区）组成的压控电流源。

## 🚀 核心算法说明
1. **多任务伪操作系统 (裸机状态机)**：
   依靠 Timer0 的 1ms 中断，维护系统时间戳（Tick）。主循环采用非阻塞式调度：数码管 1ms 刷新，按键 20ms 扫描防抖，ADC 采集与 PID(若需)/曲线刷新每 50ms 运行一次，保证屏幕显示丝滑不卡顿。
2. **多点线性校准 (Calibration)**：
   由于硬件电阻温漂、运放失调电压的存在，理论计算的 DAC 值会产生误差。本代码在 `calibration.c` 中内置了一张通过六位半万用表实测的 `[设定电流] -> [实测DAC控制字]` 映射表。设定电流时，MCU 通过线性插值算法计算下发给 DAC 的实际数据，以此满足题目 `0.1% ± 1mA` 的极高精度要求。
3. **曲线绘制 (Curve Draw)**：
   充分利用芯片 34KB SRAM 的优势，开辟一个长度为 128 的 `uint8_t` 数组记录历史电流值，每次 ADC 采集新数据后，数组整体左移，并调用屏幕的 `DrawLine` 函数连线，实现示波器般的滚动刷新效果。

## ⚙️ 编译与下载指南
1. 确保电脑已安装 Keil C251，并在 STC-ISP 中将 STC 芯片包导入到 Keil 中。
2. 双击工程文件打开，点击 `Rebuild` 或按 `F7` 进行编译，确保 0 Errors。
3. 打开 STC-ISP 软件：
   * 芯片型号选择 `AI8051U-34K64`
   * 波特率设置推荐 `115200`
   * 点击“下载/编程”后，按下普中开发板上的白色电源开关（关闭再开启，完成冷启动即可自动下载）。
